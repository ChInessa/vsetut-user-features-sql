# SQL-проект: пользовательская витрина маркетплейса «ВсёТут»

## Описание проекта

Проект выполнен в рамках кейса по аналитике данных.  
Цель — разработать пользовательскую витрину данных для маркетплейса «ВсёТут» и решить несколько ad-hoc аналитических задач на её основе.

## Задачи проекта

- определить топ-3 региона по количеству заказов;
- собрать пользовательскую витрину с метриками заказов, оплат, отмен и рейтингов;
- сегментировать пользователей по количеству заказов;
- найти пользователей с самым высоким средним чеком;
- проанализировать региональные показатели;
- изучить активность пользователей по месяцу первого заказа в 2023 году.

## Используемые навыки

- SQL;
- CTE;
- оконные функции;
- агрегатные функции;
- `case when`;
- работа с датами;
- расчет пользовательских и продуктовых метрик;
- ad-hoc аналитика.

## Основные метрики витрины

В витрину вошли следующие показатели:

- дата первого и последнего заказа;
- lifetime пользователя;
- количество заказов;
- средний рейтинг заказа;
- количество отменённых заказов;
- доля отмен;
- общая и средняя стоимость заказов;
- количество заказов в рассрочку;
- количество заказов с промокодом;
- факт использования денежного перевода;
- факт использования рассрочки;
- факт отмены заказа.

## Результаты анализа

- Большинство пользователей совершает только один заказ.
- Основная доля заказов приходится на топ-3 региона.
- Промокоды используются редко.
- Рассрочка является популярным способом оплаты.
- Пользователи с высоким средним чеком чаще совершают 3–5 заказов.
- В 2023 году наблюдалась сезонность пользовательской активности.
