# Описание проекта

## Название проекта
Разработка пользовательской витрины маркетплейса «ВсёТут»

## Цель проекта
Разработать пользовательскую витрину данных для анализа поведения клиентов маркетплейса и решить ad-hoc аналитические задачи на её основе.

## Структура проекта

### 1. Создание пользовательской витрины
В рамках проекта была разработана витрина `product_user_features`, содержащая:

- информацию о заказах пользователей;
- показатели активности клиентов;
- метрики отмен заказов;
- показатели использования рассрочки и промокодов;
- данные о рейтингах заказов;
- финансовые показатели пользователей.

Витрина построена для пользователей из топ-3 регионов по количеству заказов.

### 2. Решение ad-hoc задач
На основе витрины были выполнены аналитические задачи:

1. Сегментация пользователей по количеству заказов;
2. Ранжирование пользователей по среднему чеку;
3. Анализ статистики по регионам;
4. Анализ активности пользователей по месяцам первого заказа в 2023 году.

## Используемые технологии и навыки

- SQL
- PostgreSQL
- CTE
- оконные функции
- агрегатные функции
- CASE WHEN
- работа с датами и временем
- продуктовая аналитика
- ad-hoc анализ

## Основные метрики витрины

- количество заказов;
- средний рейтинг заказов;
- количество отменённых заказов;
- lifetime пользователя;
- общая и средняя стоимость заказов;
- количество заказов с промокодом;
- количество заказов в рассрочку;
- использование денежных переводов;
- доля отменённых заказов.

## Основные выводы

- Большинство пользователей совершают только один заказ;
- Москва лидирует по количеству клиентов и заказов;
- Рассрочка используется значительно чаще промокодов;
- Пользователи с высоким средним чеком чаще совершают 3–5 заказов;
- В 2023 году наблюдалась сезонность пользовательской активности.
